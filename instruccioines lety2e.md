# lety2E — Resumen del proyecto

## Identidad de marca

### Paleta de colores
| Color | Nombre | Hex |
|---|---|---|
| 🩷 | Magenta | `#FF00AA` |
| 🩵 | Turquesa | `#00DEC8` |
| 💜 | Morado | `#4A0080` |
| ⬛ | Fondo oscuro | `#1A0828` |
| 🟫 | Fondo cuerpo | `#F2DBD5` |
| 🟫 | Fondo tarjeta | `#FBF2EF` |

### Tipografía
- **Display / Logo**: Playfair Display Italic 900 (Google Fonts)
- **Cuerpo**: DM Sans 300/400/500 (Google Fonts)

### Logo
- **Ícono magenta**: fondo `#FF00AA` + letras `#00DEC8`
- **Ícono morado**: fondo `#4A0080` + letras `#00DEC8`
- **Wordmark oscuro**: fondo negro + `lety2E` magenta
- **Wordmark claro**: fondo blanco + `lety2E` morado
- Archivos: `generar-iconos-2e.html` y `generar-wordmarks-2e.html`
